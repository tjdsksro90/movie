### 기본 요구 기능
- jQuery 라이브러리 사용없이 **순수 바닐라 자바스크립트** 사용
- TMDB 오픈 API를 이용하여 인기영화 데이터 가져오기
- 영화정보 카드 리스트 UI 구현
- 영화 검색 UI 구헌
- 실시간 검색과 검색 버튼을 통한 검색 2개 구현
